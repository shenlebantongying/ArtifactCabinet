# Effective Matlab

## Short Cuts

Ctrl + Enter -> run section
Ctrl + R/T -> Comment/Un
Ctrl +[] -> tab level

Ctrl +
    0 -> Command window
    1 -> History
    2 -> Switch files
    Page up/Down -> next/perv tabs

Ctrl + D -> jump to defination

Ctrl + K -> delete rest of the line
Home/End -> begin/end of the lien

Alt -> GUI quick jump

F9 -> send selection to REPL
